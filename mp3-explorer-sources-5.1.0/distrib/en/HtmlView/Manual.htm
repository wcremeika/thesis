Introduction

[ mp3 - explorer ] is a plugged-in music files explorer, it helps you manage a large local collection of music files and offers direct access to musical web sites and search engines.

 

[ t o p ]

Files Localization

[ mp3 - explorer ] looks for all the MP3 files stored either on local, detached or network CD units or simply on a selected directory of your hard drive. This last type is to be preferred as it avoids the software having to scan the complete disc. In the option set-up (File\ Option…), you define favorite addresses, the search can be executed on several units. To avoid performing an extended search every time [ mp3 - explorer ] is started, a cache file is included in the program to keep an image of your music files location. When changes are made in the organization of the files, just press the F5 key to update the cache file.

 

[ t o p ]

Organization

[ mp3 - explorer ] automatically organizes your music files in several views according to a selection of criteria. Files are sorted based on information stored in ID3 tag, i.e.: author, album and the music style. Each view is a tree structure in which you navigate very much in the same way as with Windows Explorer. The three views of [ mp3 - explorer ] offer the following classification:

  • Artist / Album
  • Kind / Artist / Album
  • Album / Artist

If the ID3 tag is empty, an 'undefined' category is created, we will see that you can easily create or edit the tags.

 

[ t o p ]

File details

[ mp3 - explorer ] displays the main file details i.e.: name, location and size, information stored in the ID3 tag, and compression details (frequency, bit rate...), making it easy to pick your favorite music and control the audio quality of your files. You can restrict the information shown to a selection of theses details. In the set-up options, you can arrange the columns' display: order, size and alignment. Last but not least, you can sort the files displayed according to any of the tag criteria by double-clicking on the title of the desired column.

Edit the ID3 tag

ID3 tags are small storage blocks included in audio files, they contain information on the piece such as artist, title or genre.
Currently there is two quite different implementations of the ID3 standard.

  • The version 1 provides a limited set of information : title, artist, album, year, genre and comment (version 1.1 added the track number). All information have also a fixed lenght. This tag is saved at the end of the file. Many software are only compatible with this standard.
  • The version 2 provides a completly new system based on frames. Much more information can be stored and there is no length limitation. This standard notably supports pictures and lyrics.
    Since version 4.0, [ mp3 - explorer ] uses ID3 v2 by default. To stay compatible with other software, it can write also an old ID3 v1.1 formatted tag (see into advanced options).
    There are several ways of editing or creating ID3 tag using [ mp3 - explorer ].

    The first option is to use the 'File/ Property…' menu path (shortcut is control/ enter). By doing this, you open a dialogue box in which you can enter or edit the details of the ID3 tag fields. Music styles are presented in a drop-down list (ID3v1 compatible genres are hilighted). It is also possible to edit several tags simultaneously, in this case, the dialogue box fields are blank, the edited fields will replace the existing information. To erase fields on several tags at the same time, just place a space in the fields to be erased.

    The second option allows you to change the music style, the artist and the album simply by using the 'Drag 'n Drop' function. Just select the files and move them to the category of your choice. The tags will be edited according to where you drop the files.

    Finally, you can use the IntelliTag feature, that will detect and extract from filenames, information such as Artist, Album, Track Name and Number.

    Renaming of the files

    With [ mp3 - explorer ] you can rename your MP3 files according to information stored in the ID3 tag. For this go to the option set-up menu (Tools/ Option). Here you can define a number of preset formats based for each tag field, including punctuation and upper/lower case rules. For example Artist names always have an uppercase initial or blanks are always replaces by the underscore sign. Files can be renamed either using the rename command or when you edit the ID3 tag info.

    Play your MP3 files

    From version 4 [ mp3 - explorer ] includes its own 'player' [ mp3 - player ]. Common players such as Winamp and Sonique may also be used (available free at www.winamp.com and www.sonique.com ). Actually [ mp3 - explorer ] may work with any player on the market. There are two ways of feeding the player:

    • The 'play' command sends the files directly to the player for an immediate playing.
    • The 'enqueue' command places the files at the bottom of the playlist, without stopping the current piece.

     

    [ t o p ]

    Playlists

    The playlists are personalized selection of pieces, they may be sent to the player providing you with hours of your favorite music. It is one of [ mp3 - explorer ]'s most powerful feature. The two upper windows are dedicated to your collection of music files, the lower windows are dedicated to the management of your playlists when the playlist button of the command bar is pushed. You can easily pick pieces from the upper windows and drop them the lower windows into the standard play list or an existing one. Just save the list before you leave [ mp3 - explorer ] just as you would save any file, [ mp3 - explorer ] will save the playlist as M3U or PLS files. Playlists are stored on you disc at the address indicated in the option set-up. The content of list can be shuffled, reordered or edited to your taste using the right click of the mouse.

    The Web Window

    The forth pane is a Window open through the Web.
    When you are not connected to the Internet, here is what you can view in this window:
    - all your titles as Web pages. Information and notably image attached to your files are rendered in this page.
    - the present user's manual.
    - the readme file giving you the lastest legal and technical information about [ mp3 - explorer ].
    When you are connected, you can access to:
    - the [ mp3 - explorer ] online zone. This area will provide you personalized services.
    - selected resources such as music sites or search engines to find music on the Web.
    - resources linked in your files such as artist's web site.

     

    [ t o p ]

    Extensions (plugins)

    Plugins are add-on modules that can modify the appearance or the language of [ mp3 - explorer ] Here is the way to setup a new plugin :
    1. Download the plugin from the web site,
    2. Run the plugin installation,
    3. In the Options menu, goto Plugin and choose your new plugin in the list.
    Plugins are version-dependant.

Definitions

MP3

MP3 stands for MPEG Audio Layer-3, which is the standard method for compressing numerical audio files. This compression process was developed in Germany mainly by the Fraunhofer Institute and the University of Erlangen. The algorithm resulting from this work was standardized by the ISO (the international organization of standardization) within the framework of the recommendations made by the MPEG group in charge of the numeric video standards (Moving Picture Expert Group). The MP3 format MP3 corresponds today to ISO-MPEG standards IS 11172-3 and IS 13818-3. This compression method is based on two principles, all data related to sounds that cannot be heard by the human ear are removed, that's why it's called a destructive algorithm. The remaining data is then compressed with traditional algorithms.
